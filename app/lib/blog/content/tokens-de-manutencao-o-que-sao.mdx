Depois que seu app é entregue, o trabalho não acaba. Bugs surgem, usuários pedem melhorias, o mercado muda. É aí que os **tokens de manutenção** entram — e neste post vamos explicar como o sistema funciona na prática.

## O que é um token?

Um token equivale a **1 hora de trabalho** da equipe Fropty aplicada ao seu app. Pode ser para:

- Corrigir um bug
- Ajustar um layout
- Adicionar uma funcionalidade pequena
- Integrar uma nova API
- Melhorar uma tela existente
- Treinar sua equipe em algum fluxo

Você solicita via WhatsApp, descrevemos o que vai ser feito, confirmamos quantos tokens serão usados — e executamos.

A diferença de preço do token avulso (R$ 150 vs R$ 300) é o principal motivo para manter um plano ativo. Você economiza **50% por token** sendo assinante.

## Tokens não usados acumulam?

Não. Tokens não utilizados no mês não são transferidos para o mês seguinte — funcionam como um pacote mensal. Por isso, é importante usar bem os tokens que você tem.

**Dica:** use tokens para evoluções estratégicas, não apenas correções de emergência. Um app que cresce junto com seu negócio retém mais usuários e gera mais resultado.

## Fidelidade mínima

A fidelidade mínima é de **3 meses**. Isso garante continuidade no desenvolvimento e permite que a equipe mantenha contexto profundo sobre o seu produto.

Para cancelar ou pausar o plano, basta avisar com 30 dias de antecedência.

## Taxa de reativação

Se você cancelar e quiser reativar o plano depois, existe uma **taxa de reativação de R$ 79,90**. Ela cobre o trabalho de revisão do projeto e atualização do ambiente técnico após o período inativo.

## Como solicitar uso de tokens

1. Entre em contato via WhatsApp com o que você precisa
2. A equipe avalia e informa quantos tokens serão utilizados
3. Você aprova
4. O trabalho é executado geralmente em 1–3 dias úteis

Simples assim. Sem contratos longos, sem reuniões intermináveis — só resultado.
